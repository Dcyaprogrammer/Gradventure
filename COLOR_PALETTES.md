# 🎨 Gradventure 游戏 UI 配色方案备份库

这是在项目迭代过程中尝试过的各种 UI 配色方案备份。

## 1. Bright Macaron (明亮法式马卡龙 + 婴儿蓝)
> **状态**: 当前应用
> **特点**: 护眼、高明度、去除了脏灰感，适合长时间游玩，带有美式甜点/游乐园的轻松氛围。
> **适用**: Neo-brutalism 风格（粗黑边、硬阴影、不规则形状）。

### 核心色板 (属性对应)
- ⚪️ **背景白 (Background / Cream)**: `#FDF9F1` (透亮的暖奶油色)
- 🟣 **GPA / 学术表现 (Pastel Purple)**: `#D0BFFF` (明亮的薰衣草紫，原为黑色)
- 🔴 **心态值 / 负面反馈 / 警告 (Pastel Peach)**: `#FFA6A6` (柔和的草莓粉红)
- 🟡 **精力值 / 强调元素 / 跑马灯 (Pastel Lemon)**: `#FFE066` (饱满清透的亮黄色)
- 🔵 **经验值 / 正面反馈 / 购买 (Baby Blue)**: `#89CFF0` (清透婴儿蓝，替代了原有的薄荷绿)

### 装饰与点缀
- 🟣 **次要装饰 / 背景阴影 (Pastel Purple)**: `#D0BFFF` (明亮的薰衣草紫)
- 🌸 **卡牌装饰 / 角色名块 (Pastel Pink)**: `#FFB3D9` (娇嫩的亮粉色)

## 2. Retro Morandi Pastel (莫兰迪马卡龙)
> **状态**: 历史备份
> **特点**: 在马卡龙的基础上加入了一点点灰度（莫兰迪色系感），整体显得更高级、复古，完全没有荧光感，护眼且耐看。
> **适用**: 想要保留前卫设计感，但又希望色彩内敛、有复古高级感的场景。

### 核心色板 (属性对应)
- ⚪️ **背景白 (Background / Seashell)**: `#FDF6F5` (海贝白，微暖偏粉)
- 🟣 **GPA / 学术表现 (Lilac)**: `#E2D1F9` (灰度丁香紫)
- 🔴 **心态值 / 负面反馈 / 警告 (Salmon Pink)**: `#FFB6B9` (三文鱼粉，比草莓粉更灰一点)
- 🟡 **精力值 / 强调元素 / 跑马灯 (Vanilla)**: `#F3E5AB` (复古香草黄，不刺眼)
- 🔵 **经验值 / 正面反馈 / 购买 (Lake Blue-Green)**: `#A2E1DB` (灰湖蓝绿，莫兰迪质感)

### 装饰与点缀
- 🌸 **卡牌装饰 / 角色名块 (Dusty Rose)**: `#F2D8D8` (低饱和灰粉色)
